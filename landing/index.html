How it works

  • Install Xerolas and launch it once.
  • Use the floating widget or Ctrl+Shift+Space to capture any region.
  • Xerolas sends the image to xerolas.ideepakchauhan7.workers.dev.
  • Gemini analyzes the capture and Xerolas shows the answer beside it.
  • The app updates from the separate public downloads repo, while the source code can stay private.
